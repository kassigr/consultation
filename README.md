# Stress Management Consulting Website

This is a single-page website for a stress management consulting service, built with pure HTML, CSS, and JavaScript.

## How to Run Locally

1.  Clone or download this repository.
2.  Open the `index.html` file in your web browser.

## How to Deploy to Netlify

1.  Sign up for a Netlify account (it's free).
2.  Create a new site from a Git repository (GitHub, GitLab, etc.).
3.  Connect your repository.
4.  Netlify will automatically detect the `netlify.toml` file and deploy your site.

## How to Customize the Content

-   **Text:** Open `index.html` and edit the text in the different sections.
-   **Images:** Replace the images in the `images` folder with your own.
-   **Colors and Fonts:** Edit the CSS variables in `styles/main.css`.

## How to Add New Blog Posts

1.  Create a new HTML file in the `blog` folder (e.g., `article-4.html`).
2.  Copy the structure from one of the existing article files.
3.  Add your content.
4.  Add a new blog card to `blog/index.html` and the main `index.html` to link to your new article.
